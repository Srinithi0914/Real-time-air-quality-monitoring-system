# Real-Time Air Quality Monitoring System

## Project Description
This project monitors air quality using the MQ135 gas sensor and displays temperature, humidity, and air quality using an ESP32.

## Components Used
- ESP32 Dev Board
- MQ135 Air Quality Sensor
- DHT22 Sensor
- Jumper Wires
- Breadboard

## Software
- Arduino IDE
- GitHub

## Output
The Serial Monitor displays:
- Temperature
- Humidity
- Air Quality Value
- Air Quality Status (Good / Moderate / Poor)
